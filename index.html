160826-git基础命令

pwd 命令用于显示当前目录

git init命令把这个目录变成Git可以管理的仓库

现有的仓库 git remote add origin git@192.168.1.235:iOSTeam/Tornado-iOS.git 这个要替换项目地址

用ls -ah命令就可以看见

ssh-add ~/.ssh/zhujia360_id_rsa 添加sshKey 不至于每次输入密码

git add 文件名字.后缀 git add . // 所有文件

git status 状态

git commit -m “xxx” 提交

git log 命令查看

git log –pretty=oneline 参数 显示 commit id（版本号）

git diff 查看修改内容

git reset命令 回退到上一个版本 git reset –hard HEAD^^^^ / HEAD~4

办法其实还是有的，只要上面的命令行窗口还没有被关掉，你就可以顺着往上找啊找啊，找到那个append GPL的commit id是3628164…，于是就可以指定回到未来的某个版本：
git reset –hard 3628164

git reflog用来记录你的每一次命令

cat readme.txt 打开文件

git checkout — 可以丢弃工作区的修改 , 没有–，就变成了“切换到另一个分支”的命令，
git reset HEAD 可以把暂存区的修改撤销掉（unstage），重新放回工作区：

git rm删掉，

git push origin master 推到远程master分支
